Overview
This project is a simple personal bank account manager written in Python.
It demonstrates OOP principles using two classes:
- Amount (represents a transaction)
- PersonalAccount (represents the bank account)

Features
- Deposit and withdrawal operations
- Balance tracking
- Transaction history
- Operator overloading:
  - account + amount acts like deposit
  - account - amount acts like withdraw
- Unit tests

Files
- account.py — main classes + interactive demo
- test_account.py — unit tests
- uml diagram

Requirements
- Python 3.9+ recommended


python account.py
